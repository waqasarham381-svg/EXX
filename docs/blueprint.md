# **App Name**: EX – Exam Center

## Core Features:

- Interactive Landing Page: A polished main website featuring a hero section with a large headline, introductory subheading, and a primary login button, alongside a section highlighting platform benefits via modern cards/icons. Includes a sample exam paper display.
- Unified Login System: A modal or dedicated page presenting a single login button that leads to user type selection (Student, Teacher, Admin).
- Admin Authentication & Dashboard: Secure login logic for Admins using predefined credentials ('Arham'/'EX'), redirecting to a dedicated Admin Dashboard upon success.
- Admin Teacher Request Panel: Within the Admin Dashboard, display a panel listing teacher submissions awaiting approval, showcasing basic overview metrics like 'Total Tests Listed'.
- Admin Password Allocation: A section in the Admin Dashboard allowing the admin to input a Teacher Name/ID and assign a password, with a 'Assign Password' button.
- Teacher & Student Login Forms: Clean UI login forms tailored for both Teacher and Student roles, accessible after user type selection.

## Style Guidelines:

- Primary action color: A vibrant yet professional yellow (#FFDD00), evoking clarity and energy. This color is used for highlights and primary interactive elements.
- Background color: A subtle, clean off-white (#FDFAE7) with a faint yellow tint, providing a spacious and pristine canvas, consistent with the light theme.
- Accent color: A deep, authoritative blue (#0F5CA3) for key contrasts and secondary branding elements, representing trust and professionalism, complementing the primary yellow.
- Supportive accent: A soft, neutral gray (#B0BBC4) for borders, subdued text, and backgrounds of cards, contributing to the clean, minimal aesthetic without the starkness of black.
- Body and headline font: 'Inter', a modern grotesque-style sans-serif, chosen for its clear legibility, neutrality, and professional appeal across all textual content.
- Utilize minimalistic, outlined icons and abstract academic visuals for the hero section and features, aligning with the modern, clean, and tech-driven aesthetic.
- Implement generous spacing, soft shadows, and rounded corners throughout the UI. The overall layout will adhere to a modern SaaS/dashboard aesthetic with a clean, component-based, grid structure to maximize readability and usability.
- Incorporate subtle transition animations for elements like modal openings and navigation, and delicate hover effects on interactive components, to enhance the user experience with a polished and modern feel.
